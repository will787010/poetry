魔邪深造次，钟老细叮咛。

出室防尊口，居家悟妙经。

开窗邀月亮，乱绪咐星星。
```

## 凤凰台上忆吹箫.子岁感怀

```
庚子新元，疫魔狂舞，闭门规避瘟烟。
惜海棠春早，寂寞空妍。
焉奈孤芳自赏，羞苦涩，忍负当前。
花无语，风干粉泪，乱绪千千。
绵绵，几时可以？
邀绿水青山，拥抱晴天。
彼岸垂杨绿，荫叠青钱。
啼鸟迎春欢唱，民喜乐，欣醉桃源。
芳园内，鸡偎暖鹅，犬傍猫眠。
```

```
