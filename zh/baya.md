## 琴

```
天籁识君心，妙弦弹古今。
伯牙钟子后，何处觅知音。
```

## 棋

```
楚汉硝烟起，屯兵战火开。
当时逢敌手，放马过河来。
```

## 书

```
笔剑舞豪情，龙蛇透纸惊。
颜风藏柳骨，潇洒走人生。
```

## 画

```
笔描天下意，纸缩古今情。
指点江山秀，墨涂风月明。
```

## 诗

```
午饮笑穷思，糊涂到酉时。
以为装李白，可以吐狂词。
```

## 酒

```
红尘思玉露，借饮解千愁。
礼义廉中取，疯狂变理由。
```
## 花

```
日吻朱颜醉，风揉粉泪酣。
羞心求蜜意，托蝶作先探。
```

## 茶

```
温暖上心头，三杯可解忧。
苦甘须慢味，人世有沉浮。
```
